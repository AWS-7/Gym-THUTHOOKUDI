# IRON EMPIRE - CrossFit & Fitness Studio

A premium, high-performance website for **IRON EMPIRE**, a CrossFit and Fitness Studio based in Thoothukudi, Tamil Nadu.

## 🚀 Live Features

- **Premium UI/UX**: Modern dark-themed design with neon-red and gold accents.
- **Dynamic Preloader**: Real-time network-based loading screen with "Forging Excellence" branding.
- **Smart Navigation**: Fixed sticky navbar with a smooth side-opening drawer for mobile users.
- **Membership Plans**: Interactive 3-way toggle for Men, Women, and Cardio (Men) tariff plans (2026 Price List).
- **Transformation Gallery**: Interactive before/after showcase with auto-scrolling on mobile.
- **Facility Tour**: Masonry grid gallery featuring real gym photos and a high-definition lightbox.
- **WhatsApp Integration**: Contact form redirects inquiries directly to the official WhatsApp number (+91 63804 27781).
- **Promotion Popup**: Time-delayed (30s) special offer popup with auto-close logic.
- **Interactive Tools**: 
  - AI-powered Diet Generator.
  - BMI Calculator.
  - Member Success Metrics & Fitness Charts.
- **Social Connect**: Dedicated "Follow Us" section for Instagram and Facebook.

## 🛠️ Tech Stack

- **React**: Frontend library.
- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS**: Utility-first CSS framework for custom styling.
- **Lucide React**: For premium icons.
- **Framer Motion / CSS Animations**: For smooth transitions and scroll effects.

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
